# Project 3 - User Management REST API

## Description

This is a RESTful API built using Node.js, Express.js, MongoDB Atlas, and Mongoose. It performs complete CRUD (Create, Read, Update, Delete) operations on user data and demonstrates backend development with a cloud database.

## Technologies Used

- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- Postman

## Features

- Create a new user
- Get all users
- Get a user by ID
- Update a user
- Delete a user

## Installation

1. Clone the repository

```bash
git clone <repository-url>
```

2. Install dependencies

```bash
npm install
```

3. Start the server

```bash
node index.js
```

The server will run on:

```
http://localhost:3000
```

## API Endpoints

- POST `/users`
- GET `/users`
- GET `/users/:id`
- PUT `/users/:id`
- DELETE `/users/:id`
